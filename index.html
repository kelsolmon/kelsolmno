¿Qué entidades consideraste más importantes para nuestro modelo de datos de biblioteca y por qué?

Libro Usuario, Autor, Préstamo, Categoría,

¿Cuáles son los atributos esenciales que deben incluirse para cada entidad?

Libro: Título, ISBN, autor, editorial, cantidad disponible.

Usuario: Nombre, dirección, correo electrónico.

Autor: Nombre, apellido.

Préstamo: Fecha de préstamo, fecha de devolución.

Categoría: Nombre de la categoría.

Uso de claves primarias y foráneas para relacionar entidades, como conectar un libro con su autor.

Claves primarias y foráneas para mantener la consistencia de los datos.

Restricciones de unicidad para evitar duplicados.

Normalización adecuada, uso de índices y particionamiento de tablas para mejorar el rendimiento y la escalabilidad.
